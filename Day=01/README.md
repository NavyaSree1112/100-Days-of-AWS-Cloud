## Day 1 - Create Key Pair

### What is a Key Pair?

A Key Pair is used to securely connect to AWS EC2 instances.

### Components

- Public Key
- Private Key (.pem)

### Why We Use It?

- Secure authentication
- SSH access to EC2
- Better than passwords

### Task Details

- Key Pair Name: devops-kp
- Type: RSA
- Region: us-east-1

### Learning Outcome

Learned how AWS uses public-private key authentication for EC2 access.
